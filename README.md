# Site vitrine EFREI Paris — Département Informatique

Projet de site vitrine réalisé en **HTML5**, **CSS3** et **JavaScript** pour présenter l’EFREI Paris et son univers informatique : formations, expertises numériques, campus, contact, agenda et crédits du projet.

## Aperçu du projet

Ce site a pour objectif de proposer une vitrine claire, moderne et responsive autour de l’EFREI Paris, grande école du numérique.  
Il met en avant les formations, les domaines d’expertise, les informations de contact, les événements et les crédits de réalisation du site.

## Fonctionnalités principales

- Page d’accueil moderne avec hero, image de fond, carrousel et chiffres clés
- Page Formations avec présentation des cursus et filtres dynamiques
- Page Expertises présentant les domaines du numérique : développement, cybersécurité, IA, data, cloud
- Page Contact & agenda avec formulaire de contact et calendrier dynamique
- Page Crédits présentant les membres du projet et la répartition du travail
- Header et footer communs sur toutes les pages
- Navigation responsive avec menu burger sur mobile
- Animations d’apparition au scroll
- Formulaire avec validation JavaScript
- Liens vers les réseaux sociaux EFREI
- Design responsive adapté ordinateur, tablette et mobile

## Technologies utilisées

- **HTML5** : structure des pages
- **CSS3** : mise en forme, responsive design, animations
- **JavaScript** : interactions dynamiques
- **Aucun framework** : le projet est réalisé uniquement avec les langages web de base

## Pages du site

### Accueil

La page d’accueil présente l’EFREI Paris, les grands chiffres, les domaines du numérique et les accès rapides vers les autres pages.

### Formations

Cette page présente les formations et parcours proposés autour du numérique : Programme Grande École, bachelors, mastères, cybersécurité, data, IA, cloud et digital management.

### Expertises

Cette page met en avant les pôles de compétences du département informatique : développement logiciel, cybersécurité, intelligence artificielle, cloud, data et systèmes numériques.

### Contact & agenda

Cette page contient les informations de contact, un formulaire interactif, une section admissions et un calendrier dynamique permettant d’afficher des événements.

### Crédits

Cette page présente les membres du groupe, la répartition du travail, les technologies utilisées et les fonctionnalités développées.

## Lancer le projet en local

Aucune installation particulière n’est nécessaire.

1. Télécharger ou cloner le projet.
2. Ouvrir le dossier du projet.
3. Double-cliquer sur le fichier `index.html`.

Ou avec Visual Studio Code :

1. Ouvrir le dossier dans VS Code.
2. Installer l’extension **Live Server**.
3. Faire clic droit sur `index.html`.
4. Cliquer sur **Open with Live Server**.

## Exemple avec Git

```bash
git clone https://github.com/votre-utilisateur/votre-repository.git
```
