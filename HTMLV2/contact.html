Contact & À propos

Une page complète pour présenter le projet, les membres du groupe et permettre l'envoi d'un message.

À propos du projet

Ce site a été réalisé dans le cadre d'un projet de programmation web. L'objectif est de créer un site vitrine responsive, structuré et interactif pour présenter le département informatique de l'EFREI.

🎯

Objectif

Présenter les formations, l'équipe enseignante, les permanences et les informations utiles.

🧱

Technologies

HTML5 pour la structure, CSS3 pour le design responsive et JavaScript pour l'interactivité.

⚡

Fonctions JS

Carrousel, menu mobile, filtres, recherche, animation au scroll et validation de formulaire.
